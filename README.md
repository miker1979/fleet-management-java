# Fleet Management System (Java)

A Java-based fleet management application designed to manage vehicles, drivers, and assignments using object-oriented programming principles.

## Features
- Add vehicles with make, model, year, and driver
- View fleet inventory
- Remove vehicles by ID
- Assign drivers to vehicles
- Persistent data storage using file I/O

## Technologies
- Java
- Object-Oriented Programming (OOP)
- File Handling

## How to Run
```bash
javac Main.java Vehicle.java FleetManager.java
java Main
